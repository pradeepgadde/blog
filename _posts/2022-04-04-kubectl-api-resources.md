# Kubectl API Resources

Most of the Kubernetes resources (e.g. pods, services and others) are in some namespaces. However namespace resources are not themselves in a namespace. And some resources, such as nodes and persistentVolumes, are not in any namespace.

To see which Kubernetes resources are and aren't in a namespace, we can make use of the `kubectl api-resources` command with `--namespaced=true|false` option.
